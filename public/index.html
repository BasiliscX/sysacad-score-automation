SSA (Sysacad Score Automation)

Acerca del proyecto

SSA es una API desarrollada en Node.js utilizando Express y Puppeteer. Dado un número de universidad, legajo y contraseña, esta API ingresa al sitio web académico de la facultad con Puppeteer, realiza los clics necesarios en los botones e ingresa hasta la tabla con las calificaciones, copia en memoria los campos de la tabla y devuelve un JSON con las calificaciones del usuario.

Si los datos ingresados son correctos, recibirás un JSON con la información de tus calificaciones.

También puedes utilizar herramientas como Insomnia o Postman para enviar una solicitud POST a la dirección https://sysacad-score-automation.onrender.com/api/scores. En ambos casos, recibirás el mismo JSON con tus calificaciones.

Aclaración: No se guarda ni se roba información utilizando esta API. Puedes verificar el código y constatarlo visitando el repositorio en GitHub.

Aclaración 2: Puede que la consulta tome de 5 a 10 segundos en ser respondida, ya que estoy utilizando un servicio gratuito para alojarlo...
